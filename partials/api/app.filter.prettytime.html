prettytime
filter in module app

Description

Demo: http://firestitch.github.io/fs-angular-demos/#/prettytime

Usage

In HTML Template Binding

{{ prettytime_expression | prettytime[:round][:abr][:suffix][:type] }}

In JavaScript

$filter('prettytime')(time[, round][, abr][, suffix][, type])

Parameters

ParamTypeDetails
timeintDate

The time represented as a Date object or a Unix timestamp in seconds (when type is set to 'time')
Duration represented in seconds (when type is set to 'duration')

round
(optional)
bool

Rounds the pretty time. ie: 5d 12h (not rounded) 5d (rounded)

(default: true)

abr
(optional)
bool

Abbriviate the units of measurement. ie: 5d (abbreviated) 5 days (not abbreviated)

(default: true)

suffix
(optional)
bool

add the word 'ago' or 'from now' to the end of the pretty string

(default: false)

type
(optional)
string

Specifies if the time is a date/timestamp (time) or the duration in seconds (duration).

(default: time)
