prettytime
filter in module app

Description

Converts time to a readable format. ie: Duration 6566533 = 7 days

Usage

In HTML Template Binding

{{ prettytime_expression | prettytime[:abr][:suffix][:input] }}

In JavaScript

$filter('prettytime')([round][, abr][, suffix][, input])

Parameters

ParamTypeDetails
round
(optional)
bool

Rounds the pretty time. ie: 5d 12h (not rounded) 5d (rounded)

(default: true)

abr
(optional)
bool

Abbriviate the units of measurement. ie: 5d (abbreviated) 5 days (not abbreviated)

(default: true)

suffix
(optional)
bool

add the word 'ago' or 'from now' to the end of the pretty string

(default: false)

input
(optional)
string

Specifies if the value is a date/timestamp (time) or the duration in seconds (duration).

(default: time)
